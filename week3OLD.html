CHALLENGES

Challenge 1: Stored Numbers

Each click randomly selects a stored number and adds it to a running total.

What the program should do
  1. Create a number called score that keeps its value between clicks.
  2. Create a list (array) of stored numbers. Some numbers should be positive and some negative.
  3. When the button is clicked, run a function.
  4. Inside the function, randomly select one number from the array.
  5. Add that number to score.
  6. Display the selected number and the updated score on the page.
  7. If the score reaches a maximum or minimum value, stop the program.

Challenge 2: Guess the Word

The player tries to guess a secret word in a limited number of attempts.

Challenge 2: Guess the Secret Word (Click Choices)

The page picks one secret word. The player clicks one of three word buttons to guess it.

Logic Steps
  1. Create an array of possible words.
  2. Show three array word choices on the page (as buttons).
  3. Pick one secret word at random when the page loads and store it in a variable.
  4. Set a number of attempts (for example: 3) and store it in a variable.
  5. Show three word choices on the page (as buttons).
  6. When a word button is clicked, read the clicked word (button text or data attribute).
  7. If the clicked word matches the secret word, display “Correct” and stop the game.
  8. If it does not match, subtract 1 attempt and display “Try again” and attempts left.
  9. If attempts reach 0, display the secret word and stop the game.

Challenge 3: Dice Race

Two players roll a die each round. The player with the higher roll earns 1 point. First player to reach the target score wins.

Logic Steps

  1. Create two variables to store each player’s score.
  2. Create one button on the page (HTML)
  3. Each button click runs one round of the game.
  4. Generate a random number between 1 and 6 for Player 1.
  5. Generate a random number between 1 and 6 for Player 2.
  6. Compare the two rolls:
    • If Player 1’s roll is higher, Player 1 gains 1 point.
    • If Player 2’s roll is higher, Player 2 gains 1 point.
    • If the rolls are equal, no points are awarded.
  7. Display both dice rolls and the updated scores on the page.
  8. If a player reaches the target score, display the winner and stop the game.

Challenge 4: Mood-Based Dialogue Generator (Popup Choice)

When the button is clicked, a popup asks the user how they are feeling: happy, sad, or angry. The program selects the matching mood array and then displays one random dialogue line from that array.

Logic Steps

  1. Create three arrays of dialogue lines: one for happy, one for sad, and one for angry. Put 3 lines in each array.
  2. When the button is clicked, ask the user for a mood using prompt().
  3. Convert the user’s input to lowercase so it matches your mood words (example: toLowerCase()).
  4. Choose the correct dialogue array based on the mood:
    • If the mood is "happy", use the happy array.
    • If the mood is "sad", use the sad array.
    • If the mood is "angry", use the angry array.
  5. If the user enters something else, display a message like: “Please type happy, sad, or angry.”
  6. Randomly select one line from the chosen array (Math.random + array length).
  7. Display the mood and the selected dialogue line on the page.

Challenge 5: Random Events List

Each click generates a list of random events. Most events are common, but some are rare.

Logic Steps

  1. Create an array of common events.
  2. Create an array of rare events.
  3. Decide how many events to generate each time (for example: 5).
  4. When the button is clicked, run a function.
  5. Use a loop to generate the chosen number of events.
  6. For each event:
    • Generate a random number.
    • If the number is below a set threshold, select a rare event.
    • Otherwise, select a common event.
  7. Display the list of generated events on the page.

Challenge 6: Combo Lock

The player tries to unlock a randomly generated code.

Logic Steps

  1. Create a random secret number code (314, 296...) when the game starts (Math.random).
  2. Store the secret code so it can be checked later (variable + scope).
  3. Track how many attempts remain (variable).
  4. When the button is clicked, compare the guess to the secret code (event listener + comparison).
  5. If the user enters something that is not a three-digit number, display a message asking them to try again.
  6. Use conditionals to give feedback (if / else).
  7. End the game when the code is correct or attempts run out (conditionals + control flow).
  8. Allow the game to be restarted by resetting values (function).

Challenge 7: Generate a List of Items

Each click generates several items and adds them to a list that grows over time.

Logic Steps

  1. Create an array of common items (ex: "Pen", "Notebook", "Cable", "Sticker").
  2. Create an array of rare items (ex: "Headphones", "Tablet", "Camera").
  3. Create a global array called allItems to store everything generated so far.
  4. Decide how many items to generate per click (ex: 5).
  5. When the button is clicked, run a function.
  6. Use a for loop to generate the chosen number of items.
  7. For each loop:
    • Generate a random number.
    • If the number is below a threshold, choose a random item from the rare array.
    • Otherwise, choose a random item from the common array.
  8. Add each selected item to the allItems array.
  9. Display the newly generated items and the total number of stored items.
  10. If the list reaches a maximum length, stop generating new items.

Challenge 8: Build Combined Phrases

Each click creates several short phrases by combining words from two arrays.

Logic Steps

  1. Create an array of descriptive words (ex: "Bright", "Quiet", "Broken", "Fast").
  2. Create an array of nouns (ex: "Street", "Signal", "Window", "Room").
  3. Decide how many phrases to generate per click (ex: 6).
  4. When the button is clicked, run a function.
  5. Create an empty array called phrases.
  6. Use a for loop that runs the chosen number of times.
  7. For each loop:
    • Pick a random word from the first array.
    • Pick a random word from the second array.
    • Combine them into one string.
    • If the phrase already exists in the array, choose again.
    • Otherwise, add it to the phrases array.
  8. Display the final list of phrases on the page.

Challenge 9: Collect and Summarize Numbers

Each click adds several random numbers to a list and then summarizes the data.

Logic Steps

  1. Create a global array called numbers that starts empty.
  2. Decide how many numbers to add per click (ex: 10).
  3. When the button is clicked, run a function.
  4. Use a for loop to generate the chosen number of random whole numbers.
  5. Add each number to the global numbers array.
  6. After adding numbers, use another for loop to calculate:
    • The total of all numbers
    • The highest number
    • The lowest number
  7. Calculate the average value.
  8. Use conditionals to describe the average (ex: “High” or “Low”).
  9. Display:
    • The new numbers added this time
    • The total count of stored numbers
    • Total, average, highest, and lowest values
  10. If the list reaches a maximum length, stop adding new numbers.
