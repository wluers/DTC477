Project 2: Hangman Game (10%)

Overview of Hangman Scripts: demo

Design a Hangman game using JavaScript, HTML, and CSS. You can start with the class-generated script or create a new one using ChatGPT. The goal is to style your game with CSS and showcase how JavaScript interacts with HTML elements. Add a visual hangman figure using HTML (no Canvas), and control visibility with JavaScript. Get creative with your style and theme!
